獲取 API Key 資訊。

## 請求

```
GET /status
```

### 查詢參數

| 參數 | 類型 | 必填 | 預設值 | 說明 |
|------|------|------|--------|------|
| `page` | integer | 否 | 1 | 頁碼 |
| `limit` | integer | 否 | 20 | 每頁數量 |
| `status` | string | 否 | - | 依狀態篩選 |
| `user_id` | string | 否 | - | 依用戶篩選 |

### 請求範例

```bash
curl "https://api.example.com/v1/orders?status=pending&page=1" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 回應

```json
{
  "success": true,
  "data": [
    {
      "id": "ord_abc123",
      "user_id": "usr_xyz789",
      "status": "pending",
      "total": 250,
      "created_at": "2024-01-01T00:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 5,
    "total_pages": 1
  }
}
```
