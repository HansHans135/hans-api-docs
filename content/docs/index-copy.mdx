## 簡介

訂單服務提供訂單管理相關的 API，包括創建、查詢訂單。

## 端點列表

| 方法 | 端點 | 說明 |
|------|------|------|
| GET | `/orders/:id` | 獲取單一訂單 |
| GET | `/orders` | 獲取訂單列表 |
| POST | `/orders` | 創建訂單 |

## 訂單物件

```json
{
  "id": "ord_abc123",
  "user_id": "usr_xyz789",
  "status": "pending",
  "items": [
    {
      "product_id": "prod_001",
      "name": "商品名稱",
      "quantity": 2,
      "price": 100
    }
  ],
  "total": 200,
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}
```

## 屬性說明

| 屬性 | 類型 | 說明 |
|------|------|------|
| `id` | string | 訂單唯一識別碼 |
| `user_id` | string | 下單用戶 ID |
| `status` | string | 訂單狀態：`pending`、`paid`、`shipped`、`completed`、`cancelled` |
| `items` | array | 訂單項目列表 |
| `total` | number | 訂單總金額 |
| `created_at` | string | 創建時間 (ISO 8601) |
| `updated_at` | string | 更新時間 (ISO 8601) |

## 訂單狀態流程

```
pending → paid → shipped → completed
    ↓
cancelled
```
